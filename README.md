# Event-management-service
Spring Boot Application

## Как собрать проект
Для сборки проекта необходимо:
1. Ввести в консоль команду: **./mvnw install**
2. Введите команду: **docker build -t application .**
3. Запустить Docker
4. Ввести команду **docker-compose up -d**
